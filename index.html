Irlen Syndrome

Scotopic sensitivity

  • Perceptual processing disorder
  • Light sensitivity affect's the brain's ability to process information
  • It is not a low vision concern as much as it is cognitive, but
  • pure black text on a white background might not be ideal

You don’t know what you don’t know

Keep your content open and interoperable

  • We can make educated guesses, but in creating experiences we don't want to inadvertently lock people out
  • Give the end user the ability to self-select a solution that works best for them

Content considerations

Background images

“Don’t use CSS background images to deliver content.”

—Terrill Thompson

  • It isn't visually very pleasing, but it is legible
  • I don't know about you, but I'd rather have something ugly and usable over something pretty but illegible any day of the week

Raster images

JPEGs, PNGs, GIFs, etc.

  • Images described with pixels
  • They are staic, which means the browser does not know what content they contain
  • So it is important to assign them alt descriptions

                        /* Slightly enhance important images */
                        @media screen and (-ms-high-contrast: active) {
                          img.hero {
                            filter: brightness(1) contrast(1) saturate(1.5);
                          }
                        }
    High contrast mode media query is like any other MQ in that you can use any property you want

SVG

Scalable Vector Graphics

  • Describe imags using markup
  • And computers can read markup
  • There's three main treatments to consider for HCM

                        /* Only applies to inline SVG */
                        /* Non-interactive icons */
                        .icon-svg {
                          fill: currentColor;

File saved!

                        /* Interactive icons */
                        @media screen and (-ms-high-contrast: active) {
                          button.icon-svg {
                              background: none;

                              fill: buttonText;
